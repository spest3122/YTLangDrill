Timeline Pinner

Pin, jump, loop & skip on YouTube

⚡ Quick Start

Go to any YouTube video. Use Shift + a number key to pin the current timestamp. Press the number key alone to jump back to it.

⌨️ Keyboard Shortcuts

Shortcut Action
Shift + 0–9 Set pin at current time
0–9 Jump to pinned time
Shift + R Toggle repeat / loop
Shift + C Clear all pins

✨ Features

  • A-B Loop — Set loop start (A) and end (B) points from the pin panel
  • Skip Segments — Mark pins to auto-skip sections you don't want to watch
  • Persistent Pins — Pins are saved per video and survive browser restarts

Pins are stored locally on your device. No data is collected.
