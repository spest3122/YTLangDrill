Privacy Policy

Last updated: July 16, 2026

YTLangDrill

YTLangDrill is a Chrome extension that lets users pin timestamps, create A-B loops, and set skip segments on YouTube videos using keyboard shortcuts.

Data Collection

This extension does not collect, store, or transmit any personal data or browsing information to external servers.

All user-created pin data (timestamps, loop points, skip markers) is stored locally on your device using Chrome's built-in chrome.storage.local API. This data never leaves your browser.

Third-Party Services

This extension does not use any third-party services, analytics, tracking, cookies, or external APIs.

Data Sharing

No data is shared with any third parties. All data remains entirely on your device.

Data Retention and Deletion

Pin data is stored locally and persists until you clear it. You can delete pins for any video by pressing Shift + C while watching the video, or by clicking the "Clear" button in the pin panel. Uninstalling the extension removes all stored data.

Changes to This Policy

If this privacy policy is updated, the changes will be reflected on this page with an updated "Last updated" date. Since the extension does not collect data, significant changes are unlikely.
